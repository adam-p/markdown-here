Using Markdown Here

  1. Reload your email web page or restart your web browser.
  2. Open a new Email message.
  3. Type some Markdown into the email. If you're new to Markdown, the Markdown Here Cheatsheet will make you a pro.
  4. Render the Markdown you just wrote. You can do this in multiple ways.
    • Right-click in the email and then click "Markdown Toggle".
    • Click the button on the toolbar.
    • Press the Keyboard shortcut: CTRL+ALT+M
  5. The message should now look pretty cool. Send it to your friends!
